# GenAI Travel Planner

## Project Overview
GenAI Travel Planner is an AI-powered travel planning application built using Python and Google Gemini API. It generates a personalized travel itinerary based on the user's destination, budget, number of travel days, and interests.

## Features
- AI-generated travel itinerary
- Budget-friendly trip planning
- Tourist attraction recommendations
- Food recommendations
- Travel tips
- User-friendly interface

## Technologies Used
- Python
- Google Gemini API
- Google Colab
- Generative AI

## How to Run
1. Open the notebook in Google Colab.
2. Enter your Gemini API Key.
3. Enter destination, budget, days, and interests.
4. Get your AI-generated travel plan.
